## Running the site locally
```
npm run dev
```
Then open http://localhost:3000

## Building for production
```
npm run build
npm start
```

## Deploying
The site is ready to deploy to Vercel:
1. Push to GitHub
2. Import repo at vercel.com
3. Deploy — zero config needed
